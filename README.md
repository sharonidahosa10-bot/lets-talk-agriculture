# Let's Talk Agriculture Website

A professional website for Let's Talk Agriculture - Strategic Communications for Agriculture, Sustainability and Market Systems.

## Pages

- **Home** - Landing page
- **About** - Background and expertise
- **Work** - Portfolio and case studies
- **Podcast** - Podcast section with embedded episodes
- **Service** - Services offered
- **Contact** - Contact information and form

## How to Deploy on Hostinger

1. Download all files from this repository
2. Connect to your Hostinger account via FTP or File Manager
3. Upload all files to your public_html folder
4. Your website will be live!

## File Structure

